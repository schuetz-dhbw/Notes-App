# Notes App

An exercise project used in the Android Development lecture at DHBW Heidenheim. Students implement the app independently based on defined requirements during the lecture.

## Concepts covered
- MVVM architecture with ViewModel and StateFlow
- In-memory state management
- CRUD operations (create, read, update, delete)
- Jetpack Compose UI with dialogs and list rendering
- Timestamp handling with ZonedDateTime

## Tech stack
Kotlin, Jetpack Compose
